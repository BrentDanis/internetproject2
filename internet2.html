Clients and Servers

Computers connected to the Webare called clientsand servers

  • Clients are the typical Web user's Internet-connected devices (for example, your computer connected to your Wi-Fi, or your phone connected to your mobile network) and Web-accessing software available on those devices (usually a web broswer like Firefox or Chrome).
  • Servers are computers that store webpages, sites, apps. When a client device wants to access a webpage, a copy of the webpage is downloaded from the server ont the client machine to be displayes in the user's web broswer.

So What Actually Happens?

  1. The browser goes to the DNS server and finds the real address of the server that the website lives on.
  2. The broswer sends and HTTP request messages to the server asking it to send a copy of hte website to the client.
  3. If the server approves the client's request, the server sends the client a "200 OK" message, which means "Premission granted! Here you go!", and then starts sending the website's files to the broswer as a serives of small chunks.
  4. The broswer assembles the small chunks into a complete website and displays it to you!
